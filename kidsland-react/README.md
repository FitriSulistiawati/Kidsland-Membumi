# Kidsland React

Website Kidsland Membumi dengan struktur ringkas berisi 2 halaman utama:

1. Landing page (`/`)
2. Program (`/program`)

## Struktur Singkat

```text
src/
	App.tsx
	main.tsx
	index.css
	lib/
		contact.ts
	components/
		SiteLayout.tsx
	pages/
		HomePage.tsx
		ProgramPage.tsx
```

## Jalankan

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
