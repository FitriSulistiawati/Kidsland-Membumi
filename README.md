# Kidsland Workspace

## Struktur

- `kidsland-react/` -> aplikasi utama React + Tailwind
- `legacy-static/` -> backup versi HTML lama

## Jalankan dari root `kidsland`

```bash
npm run dev
```

## Build dari root `kidsland`

```bash
npm run build
```

Atau masuk langsung ke `kidsland-react/` jika ingin pengembangan detail komponen.
